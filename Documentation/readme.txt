sparkV code in test folder and blender simulation code in robotic arm.blend

Instructions to run the project:
1) Load test.hex onto spark V of mechanical arm.
2) Connect Xbee and USB of robotic arm into the PC
3) Open blender with terminal and run the game logic by pressing 'P' in game logic mode.
